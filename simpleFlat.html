SimpleFlat
V1.2.4

SimpleFlat is a highly customizable theme for BetterDiscord with the focus on ease of use and user customizability.



How to install:

  • Step 1, download the file by clicking the download button.
  • Step 2, go to your themes folder by navigating in discord to (Settings->BetterDiscord->Themes->Open Themes Folder)
  • Step 4, put the file in the themes folder
  • Step 5, go to discord and reload discord using
    Windows: CTRL+R
    Mac: CMD+R

    • How to customize:

  • Step 1, open the theme file and edit the values in the area labeled /*Customizable Variables*/
  • Step 2, save the file and reload discord using
    Windows: CTRL+R
    Mac: CMD+R

    • How to update:

  • Step 1, remove the current theme file from your themes folder.
  • Step 2, see How to install.
  • Note: Updaing will reset customizations, copy the current custom variables to the new theme file to save them.
